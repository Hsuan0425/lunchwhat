Shortcuts Setup 午餐簿 · 捷徑設定教學

Auto Reminder at Lunch Time · 中午自動跳出來提醒你記錄

這份教學會帶你完成三件事

  • 把「午餐簿」網頁加到 iPhone 主畫面
  • 建立一個捷徑，能抓你當下的 GPS 位置並開啟午餐簿
  • 設定自動化，每天中午 13:00 自動提醒你記錄

預計 10 分鐘內完成。完全不用寫程式。

STEP 01把網頁加到主畫面

先確定你有把 lunch.html 放在能用瀏覽器開啟的地方。最簡單的幾種方式：

  • iCloud Drive：把檔案傳上去，用 iPhone 的「檔案」App 找到它，長按後選「分享」→「透過 Safari 開啟」
  • GitHub Pages 或 Netlify Drop：免費託管靜態網頁，會得到一個網址
  • 自己的 NAS 或電腦：放到同個 Wi-Fi 下可存取的位置
重要 記下這個網址，等下捷徑要用。網址應該長得像 https://你的網域/lunch.html 這樣。

加到主畫面

  1. 用 Safari（不是 Chrome，一定要 Safari）打開你的網址
  2. 點下方中間的「分享」按鈕（方框往上箭頭的那個）
  3. 往下滑，選「加入主畫面」
  4. 取個名字，例如「午餐簿」，按「加入」

完成後主畫面就會多一個午餐簿圖示，點開會全螢幕顯示。

STEP 02建立「記錄午餐」捷徑

這個捷徑會抓你的位置、開啟網頁，並把位置傳過去讓它自動搜尋附近餐廳。

建立新捷徑

  1. 打開 iPhone 內建的「捷徑」App
  2. 點右上角「+」建立新捷徑
  3. 點上方捷徑名稱，命名為「記錄午餐」，可以選一個喜歡的顏色和圖示

加入以下動作（按順序）

01 取得目前位置 搜尋「目前位置」加入。這會取得你當下的 GPS 座標。
02 取得位置的詳細資訊（取兩次） 第一次：搜尋「取得位置詳細資訊」加入，選取項目選「緯度」，輸入來源是上一步的「目前位置」
第二次：再加一個相同動作，選取項目改選「經度」，來源一樣是「目前位置」
03 URL 搜尋「URL」加入，在文字框輸入：
https://你的網域/lunch.html?lat=[插入緯度變數]&lon=[插入經度變數]
其中 [插入緯度變數] 和 [插入經度變數] 是要點選插入的魔術變數（會是藍色膠囊狀）。
04 打開 URL 搜尋「打開 URL」加入，它會自動把上一步的 URL 當作輸入。
運作原理 網頁在網址加上 ?lat=...&lon=... 後，會自動讀取這組座標並搜尋附近餐廳。不用你手動按「搜尋」按鈕。

測試一下

捷徑建立好後，點右上角「▶ 執行」測試。

  • 第一次會要求「允許取得位置」→ 選「允許一次」或「在使用 App 時允許」
  • 會要求「允許打開 Safari」→ 允許
  • 如果網頁跳出來並且自動列出附近餐廳 → 成功了

STEP 03設定每天自動提醒

讓 iPhone 在每天中午自動跳通知，點了直接開啟記錄畫面。

建立自動化

  1. 在捷徑 App 中，點下方「自動化」分頁
  2. 點右上角「+」→「建立個人自動化」（如果是第一次建立，直接點「新增自動化」）
  3. 選擇「特定時間」
  4. 時間設成「13:00」（或你平常吃飯完的時間，例如 13:15）
  5. 下方選「每天」
  6. 點「下一步」

選擇要執行的動作

  1. 搜尋並選「顯示通知」
  2. 通知內容輸入：今天吃了什麼?
  3. 再加一個動作「執行捷徑」，選剛剛建立的「記錄午餐」
  4. 點「下一步」
  5. 把「執行前先詢問」關掉（這樣才會自動執行，不然每次都要你確認）
  6. 把「執行時通知」打開（你想收到提醒的話）
  7. 點「完成」
完成 從現在起，每天 13:00 iPhone 會自動跳通知提醒你，點下去會直接開啟午餐簿並顯示附近餐廳選項。

STEP 04進階技巧

放到主畫面的快捷位置

既然你提到想放在載具 QR Code 那一欄（今日顯示／小工具區），這裡有幾種做法：

  • 捷徑小工具：長按主畫面空白處 → 左上角「+」→ 搜尋「捷徑」→ 選一個大小加入，然後長按小工具設定要執行哪個捷徑，選「記錄午餐」。這樣從主畫面往右滑（今日顯示那一頁）就能看到並一鍵執行
  • 搭配 Widgetsmith 等第三方 App：可以做出更美觀的小工具樣式
  • 鎖定畫面捷徑：iOS 16 以上，可以把捷徑設成鎖定畫面上的小工具，不用解鎖就能按

兩階段自動化（更貼心的版本）

如果想更接近你原本想要的「偵測停留最久的位置」，可以做這樣的設計：

  1. 12:00 的自動化：靜默記錄當下位置到備忘錄（當作「中午位置」）
  2. 13:00 的自動化：讀取剛才記下的位置，打開網頁搜尋該位置附近餐廳

這樣即使 13:00 時你已經回到辦公室，還是能搜尋到 12:00 時你實際吃飯的位置附近餐廳。實作上會稍微複雜，如果需要我可以另外寫一份教學。

如果自動化沒有觸發

  • iOS 的時間自動化必須 iPhone 是解鎖或正在使用狀態才會自動執行，螢幕全黑睡眠時不會觸發
  • 改善方法：把自動化觸發時間設成你通常會看手機的時間，或打開「執行時通知」這樣會有提示音
  • 確認「設定 → 捷徑 → 允許執行捷徑」是開啟的

FAQ常見問題

為什麼點通知後網頁沒有自動搜尋附近餐廳？

可能原因：

  • URL 中的 lat= 和 lon= 參數沒有正確代入，回到捷徑檢查第 3 步的變數有沒有變成藍色膠囊狀
  • 位置權限被拒絕，到「設定 → 隱私權 → 定位服務 → Safari 網站」檢查

可以改成 12:30 執行嗎？

當然可以，自動化的時間想設幾點都行。

每天都要手動按「執行」嗎？

不用。只要 Step 03 最後把「執行前先詢問」關掉，它就會每天自動執行。第一次會跳確認，允許一次後就永遠自動了。

出國或休假怎麼辦？

可以暫時關掉自動化：捷徑 App → 自動化 → 點該自動化 → 把「啟用此自動化」關掉。回國再打開。
